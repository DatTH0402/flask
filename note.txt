How to enter an app context with Python shell:
    ctx = app.test_request_context()
    ctx.push()

How to use command with flask app:
    flask --app main_python_file_name operation(run/db) options(init/migrate/upgrade)

How to delete all records of one Model in sqlite: (In terminal, after import all objects and push context)
    db.session.query(YourModel).delete()
    db.session.commit()
